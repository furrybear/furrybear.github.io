How to use npm

软件源改为国内镜像源

npm config set registry https://registry.npm.taobao.org

Command Line Options

npm install

  • --save：安装时将包机器版本信息写入package.json的dependencies，表示发布时的依赖。

  • --save-dev：安装时将包机器版本信息写入package.json的devDependencies，表示开发时的依赖。
