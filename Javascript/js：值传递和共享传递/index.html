准确的说，JS中的基本类型按值传递，对象类型按共享传递。所谓共享传递：

'use strict'
let v1={a:'Old value'};
let v1a=v1.a;
let v2=v1;
v1a='New value';
console.log(v1);
v2.a='New value';
console.log(v1);

输出为：

{ a: 'Old value' }
{ a: 'New value' }

v1a没能改变v1.a的值，v2.a成功改变v1.a的值。

利用共享传递的机制应该能避免了传值还是传引用的问题。
