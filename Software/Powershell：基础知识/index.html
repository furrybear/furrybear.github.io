允许双击执行ps1文件

管理员权限下执行set-executionpolicy -executionpolicy RemoteSigned。

语法

函数

function doMyWork{
    ls
}

Windows PowerShell Profile

Powershell启动时运行文件，类似于Bash的bashrc和profile、Zsh的zshrc。

查看此文件位置：

$profile

一般该文件在没有创建前是不存在的，使用以下命令为当前用户创建Profile并返回文件地址：

New-Item -Type file -Force $profile

Ctrl+d退出powershell

Set-PSReadlineKeyHandler -Key ctrl+d -Function ViExit

美化

Github上的Microsoft/console里的Color Tool可以修改配色，命令：

.\ColorTool.exe -b <配色文件名>

好看的配色在mbadolato/iTerm2-Color-Schemes，比如ayu。
