命令使用学习笔记之head和tail

# 打印前5行
head -n 5 <文件名>
# 打印后5行
tail -n 5 <文件名>
# 跟踪日志，并且保持读取追加数据
tail -f <日志文件名>
# 打印后5行，跟踪日志，并且保持读取追加数据
tail -n 5 -f <日志文件名>
