设置全局代理

例如，SSR的本地代理为http://localhost:1080：

export http_proxy=http://localhost:1080
export https_proxy=http://localhost:1080

A example of mounting Samba directory by /etc/fstab

//<IP>/<Directory name> <Mount directory> cifs username=<Username>,password=<Password>,uid=1000,gid=1000,file_mode=0640,dir_mode=0750 0 0

If you want to enable the configuration after modifying it, use:

sudo mount -a

Unison

单向同步（-fat表示有个目录是Windows目录）：

unison -batch -fat -force <源目录> <源目录> <目标目录>

双向同步（-fat表示有个目录是Windows目录）：

unison -batch -fat <源目录> <目标目录>
