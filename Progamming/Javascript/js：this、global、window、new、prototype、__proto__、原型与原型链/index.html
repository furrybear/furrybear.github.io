global和window

global是Node.js的全局对象、window是浏览器端的全局对象。

this

在全局函数被全局调用时指的是全局对象（在node.js和浏览器中表现差异）。

new、prototype、__proto__、原型与原型链

参考： https://www.jianshu.com/p/dee9f8b14771 https://www.zhihu.com/question/34183746

js的对象分为普通对象和函数对象。

凡是通过new Function()创建的对象都是函数对象，其他的都是普通对象。

定律：每个对象都有__proto__属性，但只有函数对象才有prototype属性，它初始是个空对象。
