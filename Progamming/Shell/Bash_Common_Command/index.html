点和source

. hehe.sh
# 上面这条和下面这条是一样的
source hehe.sh

set -e

sh里如果有语句返回非0，则脚本停止执行并抛出错误。

但是bash里如果有语句返回非0，脚本将继续执行后续语句。

如果要求bash执行完返回非0的语句后，脚本停止，那么开头添加set -e。

pwd和PWD

pwd是命令，打印工作目录绝对路径。

PWD是环境变量，内容是工作目录绝对路径。

head/tail

# 打印前5行
head -n 5 <文件名>
# 打印后5行
tail -n 5 <文件名>
# 跟踪日志，并且保持读取追加数据
tail -f <日志文件名>
# 打印后5行，跟踪日志，并且保持读取追加数据
tail -n 5 -f <日志文件名>

cut

#取每行第3个字符、第5到7个字符
cut -c 3,5-6 <文件名>
#以冒号为分隔符
cat /etc/passwd | cut -d : -f 3,5-6
