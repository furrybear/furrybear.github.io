form表单提交数据不跳转、input标签和button标签的关系

<script>  
    function sub() {  
        //自己的处理方式，比如ajax  
    }
</script>  
<form onsubmit="sub();return false;">
    <input type="text"/>
    <!--提交按钮可以这样-->
    <input type="submit" value="提交"/>
    <!--提交按钮也可以这样-->
    <button type="submit">提交</button>
</form>

这样用form的好处是既可以享受内置的“按回车等于按submit键的”的功能，又可以自行定制按submit键后表单的行为，不会发生默认的跳转行为。
